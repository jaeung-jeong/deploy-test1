# 🔮 나의 전생은?

이름을 입력하면 AI가 당신의 전생을 스토리텔링해주는 흥미유발형 웹 서비스입니다.

기원전 15만 년부터 1980년까지, 검투사부터 아메바까지 — 당신의 전생은 무엇이었을까요?

---

## 📌 서비스 소개

사용자가 이름을 입력하면 **랜덤한 시대, 직업/존재, 사망 원인**이 조합되고, OpenAI API가 이를 바탕으로 몰입감 있는 전생 스토리를 생성합니다.

### 주요 특징
- **35개 시대** — 선사시대, 고대 이집트, 고려시대, 조선시대, 디스코 시대 등 (시대별 연도 자동 매칭)
- **300개 직업/존재** — 일반 직업뿐 아니라 동물, 무생물, 미생물, 외계인, 신화 속 존재까지
- **100개 사망 원인** — "코코넛에 맞아서"부터 "시뮬레이션 서버가 다운되어서"까지
- **AI 스토리텔링** — OpenAI GPT-4o-mini가 조합을 자연스러운 이야기로 엮어줌
- **공유하기** — 결과를 클립보드 복사 또는 모바일 공유 기능으로 전달

## 🚀 시작하기

### 1. 클론

```bash
git clone git@github.com:jaeung-jeong/deploy-test1.git
cd deploy-test1
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 환경변수 설정

```bash
cp .env.example .env
```

`.env` 파일을 열고 OpenAI API 키를 입력합니다:

```
OPENAI_API_KEY=sk-proj-여기에_본인의_키를_입력
```

### 4. 서버 실행

```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속하면 서비스를 이용할 수 있습니다.

## 🎮 사용법

1. 이름을 입력합니다
2. **"전생 알아보기"** 버튼을 클릭합니다
3. AI가 전생 스토리를 생성할 때까지 잠시 기다립니다
4. 결과를 확인하고, **"공유하기"** 버튼으로 친구에게 공유합니다
5. **"다시 뽑기"** 버튼으로 다른 전생을 확인할 수 있습니다

## 🔧 API 명세

### POST `/api/past-life`

**요청:**
```json
{
  "name": "홍길동"
}
```

**응답:**
```json
{
  "name": "홍길동",
  "era": "조선시대",
  "year": "1542년",
  "job": "암행어사",
  "death": "왕 앞에서 방귀를 뀌어서",
  "story": "AI가 생성한 전생 스토리..."
}
```

## 💡 응용 예시

### 1. 파티/모임 아이스브레이커
참가자들이 각자 이름을 입력하고 전생 결과를 공유하면서 자연스러운 대화 소재로 활용할 수 있습니다.

### 2. SNS 바이럴 콘텐츠
공유하기 기능으로 카카오톡, 인스타그램 스토리 등에 결과를 올리면 친구들의 참여를 유도할 수 있습니다.

### 3. 커플/친구 궁합 비교
두 사람이 각각 전생을 뽑고 같은 시대에 살았는지, 어떤 관계였을지 상상하며 즐길 수 있습니다.

### 4. 교육/역사 흥미 유발
"조선시대 암행어사"나 "고대 이집트 서기관" 같은 결과가 나오면 해당 시대에 대한 호기심을 자극할 수 있습니다.

### 5. 커스터마이징 확장
- `data/jobs.json`에 직업을 추가하여 특정 테마(예: 게임 캐릭터, 애니메이션 속 직업)로 커스터마이징
- `data/deaths.json`에 사망 원인을 추가하여 더 다양한 시나리오 생성
- `data/eras.json`에 가상의 시대(예: 사이버펑크 2077, 스타워즈 은하 공화국)를 추가

### 6. 오프라인 이벤트 연동
QR 코드를 생성하여 카페, 축제, 팝업스토어 등에서 방문자들이 바로 체험할 수 있도록 활용할 수 있습니다.
